# Sleep Biosensor Analysis

This repository contains the code and report for a project investigating wearable and nearable biosensors for detecting motor activity during sleep, using polysomnography (PSG) as a reference.

## Files
- `Analysis.ipynb` – Jupyter notebook containing the analysis workflow in Python  
- `Report_Biosensors.pdf` – Final project report summarising the background, methods, results, and limitations  

## Notes
- No patient or participant data is included in this repository  
- Notebook outputs have been cleared due to file size limits
